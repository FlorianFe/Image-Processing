Lineare Filter

Die zweidimensionale lineare Filterung findet oft Anwendung bei Aufgaben wie der Weichzeichnung oder der Kantenextraktion.

Der jeweilige Filter wird über seine Filtermaske identifiziert, einer Matrix aus Wichtungen und einer Kennzeichnung der Koordinate (0|0) bei der zweidimensionalen linearen Filterung.
Als Eingabe nimmt die zweidimensionale lineare Filterung eine Matrix aus Signalwerten entgegen. In der Bildverarbeitung entspricht dies einer Matrix aus Pixelwerten, die ein Bild beschreiben.

Bei der Anwendung des Filters wird nun zuerst die Filtermaske punktgespiegelt. Danach wird für jeden Pixel eine Faltung vorgenommen. Hierzu wird die Koordinate des betrachteten Pixels betrachtet [...]

  • codeCode - Filtermaske
  • codeCode - Linearer Filter

Weichzeichnungsfilter

Als Weichzeichnung bezeichnet man die bewusste Veränderung eines Bildes bei dem die Zeichnung oder der Kontrast verringert wird. Es dient der Entfernung von störendem Rauschen, wie etwa Salz- oder Pfefferrauschen. Häufig erleidet das Bild nach der Weichzeichnung einen Verlust des Schärfeeindrucks.
Box Filter

Der Box Filter bildet den arithmetischen Mittelwert der Umgebung des aktuell untersuchten Pixels und ist damit ein Weichzeichnungsfilter.

  • codeCode - Box Filter

Gauß Filter

Der Gauß-Filter ist ebenfalls ein Weichzeichnungsfilter und weist ein gutes Glättungsverhalten auf. Er orientiert sich an der gaußschen Glockenkurve.

  • codeCode - Gauß Filter

Filter zur Kantendetektion

Bei der Kantendetektion wird versucht das Bild in flächige Bereiche zu unterteilen.
Einfacher Ableitungsfilter

Der einfache Ableitungsfilter betrachtet den kommenden und den letzten Pixel, misst die Änderung zwischen ihnen und stellt sie als neuen Pixelwert dar.
